# i18n-gs

# Description

Support Google Sheets import from and export to i18n json file.

# Usage

## Init

```console
i18ngs init
```

Initialize the project with config file

## Upload

```console
i18ngs upload
```

Upload the files to google sheet (only support flat key style)

## Download

```console
i18ngs download
```

Download the files from google sheet
